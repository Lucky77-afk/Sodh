# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-20

### Added
- Initial release of Sodh Solana Explorer mobile app
- Real-time Solana blockchain data exploration
- Secure wallet connection (view-only)
- Transaction history and details
- Smart contract interaction
- Project collaboration management
- SOL and USDT token support
- Modern, dark-themed UI
- Error boundary for global error handling
- API utility with retry mechanism
- Environment configuration
- TypeScript and ESLint configuration
- Jest testing setup
- Contribution guidelines and code of conduct

### Changed
- Updated dependencies to latest stable versions
- Improved TypeScript configuration
- Enhanced error handling and reporting

### Fixed
- TypeScript configuration issues
- ESLint configuration
- Build process for both iOS and Android

### Security
- Implemented secure wallet connection
- Added proper error handling for failed transactions
- Added environment variable handling for API keys
